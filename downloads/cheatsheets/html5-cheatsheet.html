HTML5 Cheat Sheet

Code4Boy - Quick Reference Guide

Document Structure

TagDescription
<!DOCTYPE html>Declares the document type (HTML5)
<html>Root element of the HTML document
<head>Contains metadata, title, links, scripts
<body>Contains all visible page content
<meta>Metadata (charset, viewport, description)
<title>Page title shown in browser tab
<link>Links external resources (CSS, icon)
<script>Embeds or links JavaScript

Text & Headings

TagDescription
<h1> - <h6>Headings (h1 is largest, h6 is smallest)
<p>Paragraph
<br>Line break
<hr>Horizontal rule / separator
<strong>Bold / important text
<em>Italic / emphasized text
<mark>Highlighted text
<small>Smaller text
<code>Inline code
<pre>Preformatted text
<blockquote>Block quotation
<span>Inline container for styling

Links & Media

TagDescription
<a href="">Hyperlink
<img src="" alt="">Image
<video>Video element
<audio>Audio element
<source>Media source for video/audio
<iframe>Inline frame (embed external content)

Lists

TagDescription
<ul>Unordered list (bullets)
<ol>Ordered list (numbers)
<li>List item
<dl>Description list
<dt>Description term
<dd>Description detail

Forms & Input

TagDescription
<form>Form container
<input type="text">Text input field
<input type="email">Email input with validation
<input type="password">Password field (masked)
<input type="number">Number input
<input type="checkbox">Checkbox
<input type="radio">Radio button
<textarea>Multi-line text input
<select> + <option>Dropdown menu
<button>Clickable button
<label>Label for form elements

Semantic HTML5

TagDescription
<header>Page or section header
<nav>Navigation links
<main>Main content area
<section>Thematic grouping of content
<article>Self-contained content
<aside>Sidebar / related content
<footer>Page or section footer
<figure>Media with caption
<figcaption>Caption for figure

Tables

TagDescription
<table>Table container
<thead>Table header group
<tbody>Table body group
<tr>Table row
<th>Table header cell
<td>Table data cell
