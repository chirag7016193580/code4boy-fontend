Common Error Fixes

Quick solutions to the most common coding errors

.ts file not running in HTML

TypeScript files cannot be directly linked in HTML. Browsers only understand JavaScript.

Solution

Compile your TypeScript file to JavaScript using tsc filename.ts command, then link the generated .js file in your HTML.

VLC No Audio Fix

VLC media player sometimes doesn't play audio in video files due to codec or settings issues.

Solution

Go to Tools > Preferences > Audio > Reset Preferences. If that doesn't work, reinstall VLC with all codecs.

npm ERR! ERESOLVE unable to resolve dependency tree

Dependency conflicts when installing npm packages due to version mismatches.

Solution

Use npm install --legacy-peer-deps or npm install --force to bypass the conflict.
