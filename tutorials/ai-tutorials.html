2. GitHub Copilot

Step 2 — GitHub Copilot Setup & Usage

GitHub Copilot is an AI pair programmer that suggests code as you type. It's trained on billions of lines of code and can write entire functions for you.

Setup Steps:

  1. Install VS Code from code.visualstudio.com
  2. Go to Extensions (Ctrl+Shift+X) and search "GitHub Copilot"
  3. Install the extension and sign in with your GitHub account
  4. Start coding — Copilot will suggest completions automatically
  5. Press Tab to accept, Esc to dismiss, Alt+] for next suggestion

Tips for Better Copilot Suggestions:

  • Write clear function names and comments before the code
  • Add JSDoc or docstring descriptions for the function
  • Keep related code in the same file for better context
  • Use Copilot Chat (Ctrl+I) to ask questions inline
JavaScript 
// Copilot Tip: Write a detailed comment, then let Copilot complete

// Function to validate an email address using regex
// Returns true if valid, false otherwise
function validateEmail(email) {
  // Copilot will auto-suggest the implementation!
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return regex.test(email);
}

// Function to fetch user data from API and display in a card
async function displayUserCard(userId) {
  const response = await fetch(`/api/users/${userId}`);
  const user = await response.json();
  const card = document.createElement("div");
  card.className = "user-card";
  card.innerHTML = `<h3>${user.name}</h3><p>${user.email}</p>`;
  document.querySelector("#user-list").appendChild(card);
}

3. Cursor AI Editor

Step 3 — Using Cursor for Development

Cursor is an AI-powered code editor built on VS Code. It has built-in AI chat, code generation, and can understand your entire codebase.

Key Cursor Features:

  • Cmd+K (Ctrl+K) — Generate or edit code with AI
  • Cmd+L (Ctrl+L) — Open AI chat panel
  • Select code + Cmd+K — Edit selected code with AI instructions
  • @codebase — Ask questions about your entire project
  • @file — Reference specific files in your prompt
Prompts 
# In Cursor Chat (Ctrl+L), try these prompts:

"@codebase How does the authentication system work in this project?"

"@file:src/App.js Add dark mode toggle to the navbar"

"Refactor this component to use React hooks instead of class component"

# With code selected, press Ctrl+K:
"Add error handling and loading states"
"Convert to TypeScript with proper types"
"Add JSDoc comments to all functions"

4. Prompt Engineering for Coders

Step 4 — Writing Better Prompts

The quality of AI output depends heavily on your prompt. Follow these patterns for better results:

Prompts 
# Pattern 1: Role + Task + Context
"You are a senior frontend developer.
Create a responsive pricing table component
using HTML and CSS with glassmorphism design.
It should have 3 tiers: Basic, Pro, Enterprise."

# Pattern 2: Step-by-Step
"Build a todo app step by step:
1. Create the HTML structure
2. Add CSS styling with dark theme
3. Add JavaScript for CRUD operations
4. Add localStorage to persist data"

# Pattern 3: Example-Based
"Here's an example API response:
{\"users\": [{\"id\": 1, \"name\": \"John\"}]}

Write a JavaScript function to fetch this API
and display users in a card grid layout."

# Pattern 4: Constraints
"Write a CSS animation for a bouncing ball.
Constraints:
- Pure CSS only, no JavaScript
- Must work on all modern browsers
- Use CSS custom properties for customization
- Add reduced-motion media query"

7. Building Projects with AI

Step 7 — AI-Assisted Project Workflow

Here's a recommended workflow for building projects with AI assistance:

  1. Plan — Ask AI to create a project structure and feature list
  2. Design — Use AI to generate UI mockups and color schemes
  3. Code — Let Copilot/Cursor help write the code with inline suggestions
  4. Debug — Paste errors into ChatGPT for instant fixes
  5. Test — Ask AI to generate test cases for your functions
  6. Deploy — Use AI to write Dockerfiles, CI/CD configs, and deploy scripts
  7. Document — Ask AI to generate README.md and API docs
Prompts 
# Project Planning
"I want to build a portfolio website with:
- Hero section with animated background
- Projects gallery with filter
- Contact form with validation
- Dark/Light theme toggle

Create a folder structure and list all files
I need to create."

# Documentation
"Generate a professional README.md for my project.
Include: description, features, setup instructions,
tech stack, screenshots section, and license.
Project: [describe your project]"

# Deployment
"Write a Dockerfile for my Node.js Express app
that uses MongoDB. Include multi-stage build
for production optimization."
