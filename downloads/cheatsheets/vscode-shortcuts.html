These shortcuts are for Windows/Linux. For Mac, replace Ctrl with Cmd and Alt with Option.

General

ShortcutAction
Ctrl + Shift + PCommand Palette (access all commands)
Ctrl + PQuick Open (search files)
Ctrl + ,Open Settings
Ctrl + `Toggle Terminal
Ctrl + BToggle Sidebar
Ctrl + Shift + EExplorer panel
Ctrl + Shift + FSearch across files

Editing

ShortcutAction

Shift + Alt + Up/DownCopy line up/down

Shift + Alt + AToggle block comment
Ctrl + HFind and Replace
Ctrl + ZUndo
Ctrl + Shift + ZRedo
Ctrl + [ / ]Indent / Outdent line

Multi-Cursor

ShortcutAction

Ctrl + Alt + Up/DownAdd cursor above/below
Ctrl + DAdd cursor at next match
