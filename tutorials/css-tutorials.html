CSS Tutorials

1. CSS Basics - Styling Your First Element

Step 1 — What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and responsive design.

Your First CSS Styles

CSS 
/* Select all h1 headings */
h1 {
  color: blue;
  font-size: 36px;
  text-align: center;
}

/* Style all paragraphs */
p {
  color: #333;
  line-height: 1.6;
  margin: 20px 0;
}
Step 2 — Adding CSS to HTML

There are three ways to add CSS to your HTML:

1. Inline CSS (Not Recommended)

HTML 
<h1 style="color: blue; font-size: 36px;">Hello</h1>

2. Internal CSS (For Single Page)

HTML 
<head>
  <style>
    h1 { color: blue; }
    p { margin: 20px; }
  </style>
</head>

3. External CSS (Best Practice)

HTML 
<head>
  <link rel="stylesheet" href="style.css">
</head>

2. CSS Flexbox - Modern Layouts

Step 3 — Flexbox Fundamentals

Flexbox makes it easy to create flexible, responsive layouts without floats or positioning.

Basic Flexbox Container

CSS 
.container {
  display: flex;
  justify-content: center;  /* Center horizontally */
  align-items: center;      /* Center vertically */
  gap: 20px;              /* Space between items */
}

Flexbox Common Properties

CSS 
/* Parent Container */
.parent {
  display: flex;
  flex-direction: row;         /* or column */
  flex-wrap: wrap;           justify-content: space-between;
  align-items: stretch;
}

/* Child Items */
.child {
  flex: 1;                  /* Grow equally */
  order: 1;                 /* Change order */
  align-self: center;       /* Override parent align */
}

3. CSS Grid - Advanced Layouts

Step 4 — Grid System

CSS Grid is perfect for complex 2D layouts with rows and columns.

Creating a Grid Layout

CSS 
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
  grid-template-rows: auto;
  gap: 20px;
}

/* Responsive: Stack on mobile */
@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;  /* Single column */
  }
}

4. Responsive Design - Media Queries

Step 5 — Mobile-First Design

Make your website look great on all devices with media queries.

Common Breakpoints

CSS 
/* Mobile First Approach */
.container {
  padding: 10px;
  width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    padding: 20px;
    max-width: 720px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    max-width: 960px;
  }
}

/* Large Desktop */
@media (min-width: 1200px) {
  .container {
    max-width: 1140px;
  }
}
