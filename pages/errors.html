.ts file not running in HTML

You linked a TypeScript (.ts) file directly in your HTML using a <script> tag, but the browser shows errors or nothing happens. Browsers cannot execute TypeScript directly.

Solution

Step 1: Install TypeScript globally: npm install -g typescript

Step 2: Compile your TS file: tsc filename.ts

Step 3: Link the generated filename.js file in your HTML instead:

HTML 
<!-- ❌ Wrong -->
<script src="app.ts"></script>

<!-- ✅ Correct -->
<script src="app.js"></script>

VLC No Audio Fix

VLC media player plays video but there's no audio. This can happen due to codec issues, wrong audio output settings, or system audio configuration.

Solution

Method 1: Go to Tools → Preferences → Audio → Output Module → Change to "DirectSound" or "Windows multimedia Device output"

Method 2: Reset all preferences: Tools → Preferences → Click "Reset Preferences" at the bottom

Method 3: Update VLC to the latest version from videolan.org

Method 4: Check Windows Sound Settings: Right-click speaker icon → Sound Settings → Make sure correct output device is selected

npm ERR! ERESOLVE unable to resolve dependency tree

When running npm install, you get dependency resolution errors because different packages require conflicting versions of the same dependency.

Solution

Quick Fix: Run npm install --legacy-peer-deps

Alternative: Run npm install --force

Best Practice: Check which packages conflict and update them to compatible versions in your package.json.

Terminal 
# Quick fix
npm install --legacy-peer-deps

# Or force install
npm install --force

# Clean install (delete node_modules first)
rm -rf node_modules package-lock.json
npm install

CORS Error: Access-Control-Allow-Origin

When fetching data from an API, you get: "Access to fetch has been blocked by CORS policy: No 'Access-Control-Allow-Origin' header is present."

Solution

For Development: Use a CORS proxy like https://cors-anywhere.herokuapp.com/ before the API URL.

For Production: Configure your server to send proper CORS headers.

Node.js/Express: Install and use the cors middleware.

JavaScript 
// Express.js - Enable CORS
const cors = require('cors');
app.use(cors());

ModuleNotFoundError: No module named 'xyz'

Python can't find the module you're trying to import. This usually means the package isn't installed or you're using the wrong Python environment.

Solution

Step 1: Install the package: pip install module_name

Step 2: If using Python 3: pip3 install module_name

Step 3: Check if you're in the right virtual environment: which python

Terminal 
# Install a package
pip install requests

# Or use pip3
pip3 install requests

# Create a virtual environment
python -m venv myenv
source myenv/bin/activate  # Linux/Mac
myenv\Scripts\activate     # Windows

# Then install
pip install requests

IndentationError: unexpected indent

Python uses indentation (spaces/tabs) to define code blocks. Mixing tabs and spaces or inconsistent indentation causes this error.

Solution

Fix 1: Use consistent indentation — always 4 spaces (recommended) or always tabs, never mix them.

Fix 2: In VS Code, press Ctrl+Shift+P → "Convert Indentation to Spaces" to fix mixed tabs/spaces.

Fix 3: Enable "Show Whitespace" in your editor to visualize indentation.

Cannot read properties of null (reading 'addEventListener')

JavaScript tries to call a method on an element that doesn't exist in the DOM. Usually because the script runs before the HTML element is loaded.

Solution

Fix 1: Place your <script> tag at the bottom of <body>, just before </body>.

Fix 2: Wrap your code in DOMContentLoaded event.

Fix 3: Add defer attribute to your script tag.

JavaScript 
// Fix: Wait for DOM to load
document.addEventListener('DOMContentLoaded', () => {
  const btn = document.querySelector('.btn');
  if (btn) {
    btn.addEventListener('click', handleClick);
  }
});

// Or use defer in HTML:
// <script src="app.js" defer></script>

404 Error: Page Not Found

The browser can't find the file or page you're trying to access. The URL path, file name, or server configuration might be wrong.

Solution

Check 1: Verify the file path and file name (case-sensitive on Linux).

Check 2: Make sure the file exists in the correct directory.

Check 3: For relative paths, use ./ for same directory, ../ for parent directory.

Check 4: Check your server configuration for correct routing.

Git: "fatal: not a git repository"

You're trying to run a git command but you're not inside a git repository directory.

Solution

Fix 1: Navigate to your project folder: cd my-project

Fix 2: Initialize git if it's a new project: git init

Fix 3: Clone an existing repo: git clone https://github.com/user/repo.git

"Uncaught SyntaxError: Unexpected token" in JavaScript

JavaScript encountered a character or token it wasn't expecting. Common causes: missing brackets, extra commas, or typos.

Solution

Fix 1: Check for missing or extra brackets {}, parentheses (), or semicolons ;

Fix 2: Look at the line number in the error message — the issue is usually on that line or the line above.

Fix 3: Use an online JavaScript validator or your editor's linting extension (ESLint).

Fix 4: Make sure you're not using reserved keywords as variable names.
