HTML Tutorials

1. Introduction to HTML

Step 1 — What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a web page using elements and tags.

Every website you visit is built with HTML at its core. Think of HTML as the skeleton of a website — it defines what content appears on the page.

Your First HTML Document

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Website</title>
</head>
<body>
  <h1>Hello World!</h1>
  <p>Welcome to my first website.</p>
</body>
</html>
Step 2 — Common HTML Tags

Here are the most common HTML tags you'll use every day:

  • <h1> to <h6> — Headings (h1 is largest)
  • <p> — Paragraphs
  • <a> — Links
  • <img> — Images
  • <div> — Container/Division
  • <ul> / <ol> — Lists
  • <form> — Forms for user input

Creating Links and Images

HTML 
<!-- Creating a link -->
<a href="https://code4boy.com" target="_blank">
  Visit Code4Boy
</a>

<!-- Adding an image -->
<img src="photo.jpg" alt="My Photo" width="300">

<!-- Creating a list -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

2. HTML Forms & Input Elements

Step 3 — Building Forms

Forms are essential for collecting user data. They use the <form> tag along with various input types.

HTML 
<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" placeholder="Enter your name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="password">Password:</label>
  <input type="password" id="password" name="password" minlength="8">

  <label>Gender:</label>
  <input type="radio" name="gender" value="male"> Male
  <input type="radio" name="gender" value="female"> Female

  <label>Skills:</label>
  <input type="checkbox" name="html"> HTML
  <input type="checkbox" name="css"> CSS
  <input type="checkbox" name="js"> JavaScript

  <label for="country">Country:</label>
  <select id="country" name="country">
    <option value="india">India</option>
    <option value="usa">USA</option>
    <option value="uk">UK</option>
  </select>

  <label for="message">Message:</label>
  <textarea id="message" name="message" rows="4"></textarea>

  <button type="submit">Submit</button>
</form>

3. HTML Tables

Step 4 — Creating Tables

Tables are used to display data in rows and columns. They are built with <table>, <tr>, <th>, and <td> tags.

HTML 
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Sarah</td>
    <td>30</td>
    <td>London</td>
  </tr>
</table>
