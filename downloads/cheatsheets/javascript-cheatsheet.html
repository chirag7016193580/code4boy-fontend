Variables & Data Types

Functions

// Regular Function function greet(name) { return `Hello, ${name}!`; } // Arrow Function const add = (a, b) => a + b; // Default Parameters const welcome = (name = "Guest") => `Hi ${name}`; // Rest Parameters const sum = (...nums) => nums.reduce((a, b) => a + b, 0);

Array Methods

MethodDescriptionExample
.map()Transform each elementarr.map(x => x * 2)
.filter()Keep elements that pass testarr.filter(x => x > 5)
.reduce()Reduce to single valuearr.reduce((a,b) => a+b, 0)
.find()Find first matching elementarr.find(x => x > 3)
.findIndex()Find index of first matcharr.findIndex(x => x > 3)
.forEach()Loop through each elementarr.forEach(x => log(x))
.includes()Check if array contains valuearr.includes(5)
.some()Check if any element passesarr.some(x => x > 10)
.every()Check if all elements passarr.every(x => x > 0)
.sort()Sort elementsarr.sort((a,b) => a-b)
.flat()Flatten nested arrays[[1],[2]].flat()
...spreadSpread array elements[...arr1, ...arr2]

Destructuring & Spread

// Array Destructuring const [a, b, ...rest] = [1, 2, 3, 4]; // Object Destructuring const { name, age, ...others } = { name: "Ali", age: 25, city: "Delhi" }; // Spread Operator const merged = { ...obj1, ...obj2 }; const clone = [...originalArray];
